# Sizing

## Width

| クラス        | プロパティ                               |
| ------------- | ---------------------------------------- |
| `w-<number>`  | `width: calc(var(--spacing) * <number>)` |
| `w-px`        | `width: 1px`                             |
| `w-auto`      | `width: auto`                            |
| `w-full`      | `width: 100%`                            |
| `w-screen`    | `width: 100vw`                           |
| `w-dvw`       | `width: 100dvw` ⚡                       |
| `w-lvw`       | `width: 100lvw` ⚡                       |
| `w-svw`       | `width: 100svw` ⚡                       |
| `w-min`       | `width: min-content`                     |
| `w-max`       | `width: max-content`                     |
| `w-fit`       | `width: fit-content`                     |
| `w-1/2`       | `width: 50%`                             |
| `w-1/3`       | `width: 33.333%`                         |
| `w-2/3`       | `width: 66.667%`                         |
| `w-1/4`       | `width: 25%`                             |
| `w-3/4`       | `width: 75%`                             |
| `w-1/5`       | `width: 20%`                             |
| `w-2/5`       | `width: 40%`                             |
| `w-3/5`       | `width: 60%`                             |
| `w-4/5`       | `width: 80%`                             |
| `w-1/6`       | `width: 16.667%`                         |
| `w-5/6`       | `width: 83.333%`                         |
| `w-xs`        | `width: 20rem` (320px)                   |
| `w-sm`        | `width: 24rem` (384px)                   |
| `w-md`        | `width: 28rem` (448px)                   |
| `w-lg`        | `width: 32rem` (512px)                   |
| `w-xl`        | `width: 36rem` (576px)                   |
| `w-2xl`       | `width: 42rem` (672px)                   |
| `w-3xl`       | `width: 48rem` (768px)                   |
| `w-4xl`       | `width: 56rem` (896px)                   |
| `w-5xl`       | `width: 64rem` (1024px)                  |
| `w-6xl`       | `width: 72rem` (1152px)                  |
| `w-7xl`       | `width: 80rem` (1280px)                  |
| `w-[<value>]` | 任意の値                                 |

## Size (Width & Height 同時指定)

| クラス           | プロパティ                                      |
| ---------------- | ----------------------------------------------- |
| `size-<number>`  | `width/height: calc(var(--spacing) * <number>)` |
| `size-px`        | `width/height: 1px`                             |
| `size-auto`      | `width/height: auto`                            |
| `size-full`      | `width/height: 100%`                            |
| `size-screen`    | `width/height: 100vw / 100vh`                   |
| `size-min`       | `width/height: min-content`                     |
| `size-max`       | `width/height: max-content`                     |
| `size-fit`       | `width/height: fit-content`                     |
| `size-1/2`       | `width/height: 50%`                             |
| `size-[<value>]` | 任意の値                                        |
