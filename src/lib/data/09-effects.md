# Effects

## Box Shadow

| クラス | 説明 |
|--------|------|
| `shadow-2xs` | 極小シャドウ (`0 1px rgb(0 0 0 / 0.05)`) ⚡ |
| `shadow-xs` | 超小シャドウ ⚡ |
| `shadow-sm` | 小シャドウ |
| `shadow` | デフォルトシャドウ |
| `shadow-md` | 中シャドウ |
| `shadow-lg` | 大シャドウ |
| `shadow-xl` | 特大シャドウ |
| `shadow-2xl` | 超大シャドウ |
| `shadow-none` | シャドウなし |
| `shadow-<color>-<shade>` | シャドウカラー |
| `shadow-<color>-<shade>/<opacity>` | 透明度付きシャドウカラー |
| `shadow-[<value>]` | 任意の値 |
